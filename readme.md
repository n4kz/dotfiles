### Synopsis

```bash
	# Clone this repo
	git clone https://github.com/n4kz/dotfiles
	cd dotfiles

	# Init submodules
	git submodule init

	# Update
	git submodule update

	# Setup symbolic links
	stow vim
	stow bash
	stow git
```

Read more about GNU stow [here](http://www.gnu.org/software/stow/).

Enjoy
